===========
minkfncts2d
===========

An implementation of the 2D Minkowski Functionals in Python 3.4/Cython.

Based on  http://iopscience.iop.org/article/10.1088/1742-5468/20

Requirements
============

* Python 3.4
* Cython >= 0.28.4
* Numpy

Installation
============

.. code-block:: bash

    $ python setup.py install
